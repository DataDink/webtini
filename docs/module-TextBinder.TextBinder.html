TextBinder.TextBinder()

new TextBinder()

Enables the use of {data} binders in text content.

Example
```html
<html>
 <body>
  <h1>{title}</h1>
  <p>{content}</p>
 </body>
</html>
```

```javascript
var model = { title: 'Hello', content: 'World' };
var binder = new Binder(new TextBinder());
binder.bind(document.body, model);
```
